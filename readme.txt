这个文件夹中包含一些常用的，可以复用的脚本和样式的代码，以及自己写的和从网上收集的动画实例，用以提升开发效率或者作为参考实现一些动画效果, 谷歌浏览器扩展程序。

文件夹结构：

1、 custom elements
   web components实现的自定义标签，直接在页面上写入相应标签即可实现相应效果，无需引入第三方库
   
2、 demos
   自己在项目中实现过的动画以及从网上收集的动画
   
3、 javascript
   比较常用的可以复用的脚本
   
4、 stylesheets
   比较常用的可以复用的样式
   
5、 chrome extensions
   谷歌浏览器扩展程序
   
6、 test
   相应功能的测试文件
